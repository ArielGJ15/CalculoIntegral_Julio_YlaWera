Ejercicios de integrales definidas

    -Resuelve las siguientes integrales:

  1. \(\int_{-2}^2 {x^2} + 5 \, dx = \)

    2. Para resolverlo sumaremos más uno al exponente y el resultado lo usamos para dividir; para el 5 solo le multiplicamos por “x”. de la siguiente manera:

    \[\frac{x^{2+1}}{2+1} + 5x\, = \frac{x^3}{3} + 5x\]

    Y solo cambiamos x por cada limite de integracion:

    \[\frac{2^3}{3} + 5(2)\, - \frac{(-2)^3}{3} + 5(-2)\, = \frac{76}{3} = 25.33 U^2\]

  2. \(\int_{-1}^1 \frac{5}{x+2} + 3e^x \, dx = \)

    3. Cuando tenemos un numero natural siendo dividido por una función sin exponentes entonces ya sabemos que su integral es ln y lo mismo para \(e^x:\)

    \[5ln \lvert x+2\rvert + 3e^x\, = \]

    Y solo cambiamos x por cada limite de integracion:

    \[[5ln \lvert 1+2\rvert + 3e^1] - [5ln \lvert -1+2\rvert + 3e^{-1}]\, = ln \lvert 243\rvert = 5.49 U^2\]

  3. \(\int_1^4 3x^2 -5x +8\,dx = \)

    4. Para resolverlo les sumaremos más uno al exponente y el resultado lo usamos para dividir; para el 8 solo agregamos “x”:

    \[\frac{3x^{2+1}}{2+1} - \frac{5x^{1+1}}{1+1} + 8x\, = x^3 - \frac{5x^{2}}{2} + 8x\]

    Y solo cambiamos x por cada limite de integracion:

    \[[4^3 - \frac{5(4)^{2}}{2} + 8(4)]-[1^3 - \frac{5(1)^{2}}{2} + 8(1)]\, = \frac{99}{2} = 49.5 U^2\]

  4. \(\int_3^3 \frac{5}{3} {x^4} + \cos(x)\, dx = \)

    5. La integral de "\(\cos(x)\)" es: "\(\operatorname{sen}(x)\)" entonces solo resolvemos como:

    \[\frac{5}{3} * \frac{x^{4+1}}{4+1} + \operatorname{sen}(x)\, = \, \frac{5}{3} * \frac{x^5}{5} + \operatorname{sen}(x) \]

    Y solo cambiamos x por cada limite de integracion:

    \[[\frac{5}{3} * \frac{3^5}{5} + \operatorname{sen}(3)] - [\frac{5}{3} * \frac{3^5}{5} + \operatorname{sen}(3) \,] = 0\]

  5. \(\int_{-3}^1 \frac{3}{4x+5} + 5 \, dx = \)

    6. Para resolverlo separamos los reales y resolvemos por inmediata, de la siguiente manera:

    \[\frac{3}{4} ln \lvert {x+5}\rvert +5x \, = \]

    Y solo cambiamos x por cada limite de integracion:

    \[[\frac{3}{4} ln \lvert {1+5}\rvert + 5(1)] \, - \,[\frac{3}{4} ln \lvert {-3+5}\rvert + 5(-3)] \, = \, ln \lvert {3^{\frac{3}{4}}}\rvert + 20 = 20.82 U^2\]
